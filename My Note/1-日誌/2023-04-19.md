####  React Hook Form： 表單驗證方法
```js
{...register('username', { required: { value: true, message: '給我填哦'} })}

{...register('email', {
	required: {
		value: true,
		message: 'Email 為必填'
	},
	pattern: {
		value: /^\S+@\S+$/i,
		message: 'Email 格式不正確'
	}
})}

```
