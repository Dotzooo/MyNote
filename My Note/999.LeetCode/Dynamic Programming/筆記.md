## 時間序列型

	- 定義：
		- dp[i][j]：表示第 i 輪的 第 j 種狀態(j = 1,2, ...,k)
		- 將 dp[i][j] 與前一輪的狀態 dp[i-1][j] 產生關係
		- 最終結果，dp[last][j] 中的某一種 aggregation(sum, max, min)
		  
	- 套路 I：
		- 狀態與前一個時間序列的狀態有關，可能是一維（序列）或二維（狀態數量） DP
